This is a practice project that I performed during my studies in order to apply some analysis and visualization techniques to a data set.

My objective is to analyze the data and determine the data science topics most asked about by users of this site.

  • This is a site that is dedicated to data science and any of the disciplines that contribute to data science in general, including python programming, machine learning, data wrangling, statistics...etc.

  • This site also divides into several other sections including: Users Tags Unanswered questions

  • we can use the tagging system to count the number of questions per topic

  • Each questions contains indicators showing the number of views, number of answers, number of votes as well as tags which can help categorize the topics relevant to the question.

We can see that some tables are of particular importance:¶

  • The Posts table - which contains the posts, information about the users, number of votes, number of answers, number of comments,

Extracting the necessary columns from the Posts table:¶

  • We run the below querey in the DSSE

SELECT Id, PostTypeId, CreationDate, Score, ViewCount, Tags, AnswerCount, FavoriteCount FROM Posts WHERE PostTypeId = 1 AND YEAR(CreationDate) = 2019

Counting Tag Use and Tag Views:¶

In [7]:
tag_use = {}
tag_views = {}

for i in range(0, questions.shape[0]):
    tags = questions.iloc[i]['Tags']
    tag_view = questions.iloc[i]['ViewCount']
    for tag in tags:
        if tag in tag_use:
            tag_use[tag] +=1
        else:
            tag_use[tag] = 1
        
        if tag in tag_views:
            tag_views[tag] += tag_view
        else:
            tag_views[tag] = tag_view

In [25]:
def deeplearningclassify(tags):
    for tag in tags:
        if tag in ['machine-learning', 'deep-learning', 'neural-network', 'keras', 'tensorflow', 'scikit-learn']:
            return 1
    return 0
In [26]:
